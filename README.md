## 🚀 使用方法

### ローカル環境で開く

1. **ブラウザで直接開く**
   ```bash
   # ファイルをダブルクリック、または
   open index.html  # macOS
   start index.html # Windows
   ```

2. **ローカルサーバーを使用（推奨）**
   ```bash
   # Pythonを使う場合
   python3 -m http.server 8000

   # Node.jsのhttp-serverを使う場合
   npx http-server
   ```

   その後、ブラウザで `http://localhost:8000` にアクセス

## 🎯 マーケティング戦略

このサイトはPDFの「パーセプションフロー調査」に基づき、以下の要素を実装しています：

- **フレームブレーカー**: 「お墓は自分で世話すべき」という固定観念を打破
- **ベネフィット訴求**: 心の負担軽減、時間節約、プロの品質
- **購買口実**: 交通費との比較、高齢者の安全、長期的なお墓の保全
- **期待超越**: 写真報告、メッセージ代読などの付加価値
- **自己イメージ確立**: 親孝行・先祖孝行をアピール

## 🛠 カスタマイズ方法

### 色の変更
`styles.css` の `:root` セクションで色を変更できます：

```css
:root {
    --color-primary: #2d7a4f;      /* メインカラー */
    --color-primary-dark: #1e5438; /* 濃いグリーン */
    --color-primary-light: #4a9d6f; /* 薄いグリーン */
}
```

### 電話番号の設定
`index.html` の以下の部分を実際の電話番号に変更：

```html
<p class="contact-phone">0120-XXX-XXX</p>
```

### フォーム送信先の設定
`script.js` の `initContactForm()` 関数内で、実際のバックエンドAPIに接続：

```javascript
// 実際のAPI呼び出しに置き換える
fetch('/api/contact', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(formData)
})
```
